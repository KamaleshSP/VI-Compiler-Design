### Lab 0 â€“ String Operations in C (Practice Session)

**Purpose:**  
Lab 0 is a **practice lab** designed to refresh essential C programming concepts required for compiler design experiments.<br>

**Programs Covered:**
- String Reversal
- String Length Calculation
- String Comparison

**Concepts Reinforced:**
- Character arrays
- String handling
- Built-in vs manual logic
- Loops and conditionals

### Lab 1 â€“ Lexical Analyzer in C

**Purpose:**  
Lab 1 marks the **beginning of actual compiler design experiments** and focuses on implementing a **Lexical Analyzer**, which is the **first phase of a compiler**.<br>

**Program Covered:**
- Identification of tokens such as:
  - Keywords
  - Identifiers
  - Operators
  - Integers
  - Delimiters

**Key Concepts:**
- Lexical analysis
- Tokenization
- Pattern recognition
- String and character processing
